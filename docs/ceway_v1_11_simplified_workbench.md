# 策维 v1.11 精简选号工作台

## 产品主线

页面只保留三个主入口：选号工作台、当期复盘、历史数据。

选号工作台包含：

- 智能推荐：基于完整历史库的冷热、遗漏、奇偶、大小、和值与结构分散规则。
- 随机生成：不读取评分，独立随机生成单式、复式或胆拖。
- 套餐模拟：按公开套餐结构随机展开票面组合与赠票。
- 自选号码：用户自行选择单式、复式或胆拖号码。

所有结果汇入本期号码组合，可独立保存，开奖后在当期复盘逐方案核对。

## 数据底座

- 大乐透：07001 至当前最新期。
- 双色球：2003001 至当前最新期。
- 本地 SQLite 与发布快照均超过 100 期，自动更新程序继续按开奖日运行。

## 规则边界

- 大乐透单式、复式和胆拖支持追加投注，按 3 元/注计算。
- 双色球不提供追加投注。
- 套餐模拟按公开套餐结构展开，不支持自行追加。
- 热冷、遗漏与结构指标只用于历史筛选和复盘，不代表提高未来中奖概率。
