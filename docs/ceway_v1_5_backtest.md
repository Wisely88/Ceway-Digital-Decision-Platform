# 策维（Ceway）数字决策平台 v1.5 回测验证版

## 开发范围

v1.5 在 v1.4 数据底座之上补齐历史回测模块，用于验证当前决策链的历史匹配表现。

已完成：

- 历史开奖滚动回测；
- CBGO 当前评分与组合生成规则复用；
- 随机选号对照组；
- 命中记录率、最佳命中、最佳奖级统计；
- 平均前区/后区命中统计；
- Dashboard 历史回测面板；
- `GET /backtest/dlt` API。

## 接口

```http
GET /backtest/dlt?budget=20&strategy=balanced&periods=100&window=100
```

返回字段：

- `config`：预算、策略、回测期数、统计窗口和期号区间；
- `summary`：CBGO 策略回测汇总；
- `baseline`：随机选号对照汇总；
- `items`：最近回测明细；
- `disclaimer`：风险说明。

## 验证边界

回测只说明当前规则在历史数据上的匹配统计，不预测未来开奖，不承诺提高中奖概率。

ROI、奖金估算和资金曲线将在后续版本引入更严格的奖级金额模型后再开放。
