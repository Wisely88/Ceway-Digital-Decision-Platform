# 数据导入说明

v1.2 Baseline 不自动下载开奖数据，也不接入第三方 API。

走势分析全部基于本地 CSV：

```csv
issue,date,f1,f2,f3,f4,f5,b1,b2
2025001,2025-01-01,3,7,18,22,31,4,11
```

字段说明：

- `issue`：期号。
- `date`：开奖日期。
- `f1` 到 `f5`：大乐透前区号码，范围 1-35，不重复。
- `b1` 到 `b2`：大乐透后区号码，范围 1-12，不重复。

使用方式：

1. 准备最新大乐透历史开奖 CSV。
2. 进入 DLT Module。
3. 点击“导入最新CSV”。
4. 系统校验 CSV 后写入 `backend/data/dlt_history.csv`。
5. Dashboard 自动基于新 CSV 重新计算冷热号、遗漏、奇偶比、大小比、和值和评分。

如果未导入最新 CSV，系统会继续使用内置示例数据，页面会显示“本地 CSV / 示例数据”提示。
